# Section 1: For the Technical Adjacent

## Animation: `01-IntroAnimation`

CLI "getting started" welcome screen with typing effect.

## Design Notes

**Recommendation:** A simple CLI-style intro animation that sets the tone for the guide.

**Visual Concept:**
- Terminal window with "getting started" command
- Animated typing effect
- Welcome message introducing the technical adjacent audience
- Sets up the CLI aesthetic used throughout

**Duration:** ~15-20 seconds

**Key Elements:**
- Terminal header with window controls
- Green monospace text on dark background
- Blinking cursor
- Subtle glow effects

## Content Summary

This section introduces the target audience - technical adjacent professionals (UX researchers, product managers, designers, data analysts) who work with technical systems but aren't developers. Establishes the "gap" between knowing AI exists and using it effectively.
